# Authentication API

RESTful Authentication API built with **Go (Golang)**, **Gin**, and **PostgreSQL**.  
This project implements a simple authentication system using a layered architecture (Handler в†’ Service в†’ Repository) with manual Dependency Injection.

## Features

- User Registration
- User Login
- Password hashing using bcrypt
- PostgreSQL database integration
- Database migration
- Layered Architecture
- Manual Dependency Injection

# Prerequisites

- Go 1.24+
- PostgreSQL
- golang-migrate

Install migrate:

```bash
go install -tags 'postgres' github.com/golang-migrate/migrate/v4/cmd/migrate@latest
```

---

# Installation

Clone repository

```bash
git clone <repository-url>
```

Masuk ke project

```bash
cd backend1
```

Install dependency

```bash
go mod tidy
```

---

# Environment

Buat file `.env`

```env
DATABASE_URL=postgres://postgres:password@localhost:5432/backend1?sslmode=disable
```

Sesuaikan dengan database PostgreSQL milikmu.

---

# Database Migration

Migration Up

```bash
migrate -path migrations \
-database "$DATABASE_URL" up
```

Migration Down

```bash
migrate -path migrations \
-database "$DATABASE_URL" down
```

---

# Run Application

```bash
go run cmd/main.go
```

Server berjalan pada

```
http://localhost:8080
```
